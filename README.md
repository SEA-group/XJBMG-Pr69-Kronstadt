# Battlecruiser Kronstadt model swap mod for Moskva

======================Presented by SEA group======================



Installation: 

Copy "PnFMods" and "PnFModsLoader.py" into /res_mods/a.b.c.d(current version)/


安装：
把PnFMods和PnFModsLoader.py放到
游戏安装目录/res_mods/版本号/ 
里面即可
